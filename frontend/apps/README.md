## How to run Apps Frontend Project?

- Copy env-sample file to .env then you can update that .env file

```bash
cp env-sample .env

# Exm: edit this lines
REACT_APP_FOOTER_MSG = "Apps Footer 2021"
REACT_APP_API_URL = "http://localhost:8000/api"

```

---

- Run project

```bash
# Install dependencies
yarn install

# Run project
yarn start

```
